想让页面效果更好，手写效果是最基本的。有一些出色的效果库可以使用，有种站在巨人肩膀上的感觉。
<!--more-->

## Canvas和SVG 绘图类 ##
- [bonsaijs](http://bonsaijs.org/)
    
    A lightweight graphics library with an intuitive graphics API and an SVG renderer.

- [kineticjs](http://kineticjs.com/)
    
    KineticJS is an HTML5 Canvas JavaScript framework that enables high performance animations, transitions, node nesting, layering, filtering, caching, event handling for desktop and mobile applications, and much more.

- [Two](http://jonobr1.github.io/two.js/)
    
    Two.js is a two-dimensional drawing api geared towards modern web browsers. It is renderer agnostic enabling the same api to draw in multiple contexts: svg, canvas, and webgl.

- [Paper](http://paperjs.org/)
- [goo.js](http://www.storminthecastle.com/projects/goo.js/)  

## animations 插件 ##
- [Animate.css](http://daneden.me/animate/)
- [Transit](http://ricostacruz.com/jquery.transit/)
    
    jQuery插件，配合Animate使用

- [Move.js](http://visionmedia.github.io/move.js/)
    
    JS写CSS动画效果
- [anima.css](http://lvivski.com/anima/)
    
    结合CSS+JS的动画库
